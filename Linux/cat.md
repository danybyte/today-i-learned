### The cat command in Linux (short for concatenate) Its primary job is to read files sequentially and write their content to standard output

# Basic Functions of cat:
1.View a File's Content:

Display the entire contents of a file directly in the terminal.
```bash
cat filename.txt
```

2.Combine Multiple Files:

Read multiple files one after another and output them as a single stream.
```bash
cat file1.txt file2.txt file3.txt
```
